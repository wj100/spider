Object.entries( obj )

返回一个给定对象自身可枚举属性的键值对数组,其排列与使用 for…in循环遍历该对象时返回的顺序一致（区别在于for…in循环也枚举原型链中的属性）
通俗讲就是Object.entries()可以把一个对象的键值以数组的形式遍历出来，顺序和for…in一致，但不会遍历原型属性
描述
Object.entries()返回一个数组，其元素是与直接在object上找到的可枚举属性键值对相对应的数组。 属性的顺序与通过手动循环对象的属性值所给出的顺序相同
JavaScript Demo: Object.entries()

const obj = {
  A:'hy',
  B:12
};
console.log(Object.entries(obj)) // [[‘A’,'hy'],[‘B',12]]

for(let [key, value] of Object.entries(obj)){
     console.log(`${key}:${value}`) // A:hy  B:12
}

将Object转换为Map

const obj = {
  A:'hy',
  B:12
};
const map = new Map(Object.entries(obj))
console.log(map) // Map { A:’hy’ , B:12 }
